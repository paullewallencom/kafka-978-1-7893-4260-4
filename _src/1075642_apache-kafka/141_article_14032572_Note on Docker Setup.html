Note on Docker Setup

Docker is notoriously hard to setup on some machines, and can have obscure bugs. If you don't know Docker at all, I recommend you first learn a bit about it online.

Running Kafka on Docker is even harder, and requires some deep understanding of Docker and Kafka.

Thankfully, IÂ have created a project to make things easy for beginners: https://github.com/simplesteph/kafka-stack-docker-compose

Note to Windows users: Docker is really hard to setup on your machines, may require Windows 10, might require disabling Hyper-V, and tinkering VirtualBox. If you cannot setup Docker, the best help you will get will be on Google.

Overall for all Windows users but Windows 10, I recommend not to run Kafka in Docker, and use the binaries instead
