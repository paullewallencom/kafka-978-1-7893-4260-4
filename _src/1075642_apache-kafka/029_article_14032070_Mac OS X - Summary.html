In summary, for Mac OS X

  1. Install brew if needed: /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

  2. Download and Setup Java 8 JDK:

    brew tap caskroom/versions
brew cask install java8

  3. Download & Extract the Kafka binaries from https://kafka.apache.org/downloads

  4. Install Kafka commands using brew: brew install kafka

  5. Try Kafka commands using kafka-topics (for example)

  6. Edit Zookeeper & Kafka configs using a text editor

    1. zookeeper.properties: dataDir=/your/path/to/data/zookeeper

    2. server.properties: log.dirs=/your/path/to/data/kafka

  7. Start Zookeeper in one terminal window: zookeeper-server-start config/zookeeper.properties

  8. Start Kafka in another terminal window: kafka-server-start config/server.properties
